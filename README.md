# Chrome Profiles Synchronizer (Windows)

Требования: Python 3.11+, Windows 10/11 (запуск от администратора).
Установка:
  python -m venv .venv
  .venv\Scripts\pip install -r requirements.txt

Запуск:
  .venv\Scripts\python src/main.py

Параметры по умолчанию в `src/app/config.py`.
Профили хранятся в `profiles/` (рядом с проектом). Каждый профиль — отдельная папка.

Функции MVP:
- Список профилей, старт/стоп, создание/удаление
- Привязка PID/HWND запущенных окон
- Выбор «Мастер»
- Глобальные хуки клавиатуры/мыши
- Доставка кликов/колеса и базовых клавиш во вторичные окна

Замечания:
- Для корректной работы хуков и отправки сообщений требуются права администратора.
