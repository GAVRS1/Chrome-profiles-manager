# Установка расширения (MV3)

1) Откройте `chrome://extensions`, включите **Developer mode**.  
2) Нажмите **Load unpacked** и выберите папку `extension/`.  
3) Откройте **Details → Extension options** и задайте:
   - Profile name: имя профиля в вашем GUI (например, prof1)
   - WS URL: `ws://127.0.0.1:8765`
4) Повторите для каждого профиля Chrome (одно и то же исходное расширение, но настройки хранятся **внутри профиля**).
5) В GUI назначьте мастер-профиль.  
6) Откройте сайт в каждой вкладке. Ввод/навигация мастера зеркалируются у ведомых.

Примечания:
- Не пробрасывайте WS наружу. Работает локально.
- Дублирование `ethereum.request` запускает те же запросы во всех профилях; кошельки могут требовать жест пользователя.
